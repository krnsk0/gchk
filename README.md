## What is this?

Interactive CLI tool. Shows a list of local branches; you pick which one to check out.

## Installation

Requires `node` > v18

```bash
npm i -g gchk
```

## Usage

You'll need a working version of `git` in your path.

To start:

```bash
gchk
```
